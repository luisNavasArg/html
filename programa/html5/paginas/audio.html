Audio

Es muy similar a la etiqueta de video en cuanto a los atributos 
  • controls es para que aparezcan los controles en la pantalla en el momento que cargue la página.
  • autoplay es usado para que apenas cargue la página se reproduzca de forma automática.
  • loop se usa para que se repita después de finalizar el audio. 
  • Ejemplo:
<!DOCTYPE Html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Audio</title>
    </head>
    <body>
    <audio controls autoplay loop>
        <source src="uno.ogg" type="audio/ogg">
        <source src="uno.mp3" type="audio/mp3">
            Tu navegador no soporta la etiqueta de audio de html5.
    </audio>
    </body>
</html>


Aparte en la etiqueta source lleva también la ruta donde se encuentra ubicado el archivo a reproducir.
Cuando hablamos de la ruta las mismas pueden ser:
  • Absolutas: "file:///home/canaima/Escritorio/index.html". 
  • Relativas:audio "/uno.mp3".
 Otra manera de usarlo es con botones para reproducir, pausar, subir o bajar volumen de esta manera:

<audio id="audio1" src="uno.mp3"></audio>
      <div>
           <button onclick="document.getElementById('audio1').play()">
               Reproducir el Audio
           </button>

           <button onclick="document.getElementById('audio1').pause()">
               Pausar el Audio
           </button>

           <button onclick="document.getElementById('audio1').volume+=0.1">
               Aumentar el Volumen</button>

           <button onclick="document.getElementById('audio1').volume-=0.1">
               Disminuir el Volumen
           </button>
     </div>

Aca te muestro como se visualiza en el navegador ambos casos
